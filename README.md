# 影子贪吃蛇 Shadow Snake

一个基于 **HTML + CSS + JavaScript Canvas** 制作的网页小游戏。
玩家需要控制贪吃蛇吃掉红宝石，同时躲避由自己过去路线形成的“影子障碍”。随着分数提升，游戏会逐步解锁新的外观、速度等级、道具和回溯机制。

## 游戏特色

* **影子障碍机制**
  玩家吃到食物后，近期移动路线会先变成预警影子，随后凝结为实体障碍。

* **多阶段外观进化**
  分数提升后会解锁不同蛇身外观，例如冰蓝核心、蓝绿电光、粉色霓虹、棱镜霓虹、黄金电弧等。

* **多种道具系统**
  游戏包含清影、超频、棱彩、磁铁、凝核等道具，不同道具拥有不同功能。

* **回溯机制**
  达到一定分数后可以获得影子回溯机会，死亡时可自动回到数秒前。

* **移动端适配**
  支持手机端浏览器游玩，包含触控方向键 / 滑动控制等移动端操作方式。

* **霓虹视觉风格**
  使用 Canvas 绘制蛇身、影子、粒子、发光和动态效果，整体采用暗色赛博霓虹风格。

## 道具说明

| 道具 | 效果                    |
| -- | --------------------- |
| 清影 | 清除周围影子障碍              |
| 超频 | 提升移动速度，并在生效期间获得特殊保护效果 |
| 棱彩 | 短时间提升红宝石得分            |
| 磁铁 | 自动吸收附近红宝石             |
| 凝核 | 缩短蛇身，扩大活动空间           |

## 进化与回溯

游戏会根据分数逐步提升速度和外观等级。
在达到特定分数后，玩家会解锁“影子回溯”能力，死亡时可以自动回退到几秒前，帮助玩家继续挑战更高分。

## 部署方式

本项目是纯静态网页，可以直接部署到：

* GitHub Pages
* Netlify
* Vercel
* Cloudflare Pages

GitHub Pages 部署方式：

1. 创建 GitHub 仓库
2. 上传 `index.html`、`style.css`、`script.js`
3. 进入仓库 `Settings`
4. 找到 `Pages`
5. Source 选择 `Deploy from a branch`
6. Branch 选择 `main`，目录选择 `/root`
7. 保存后等待生成网页链接
